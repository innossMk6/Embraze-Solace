# Embrace Solace — Trading Analysis Tool

**Embrace Solace** is a non-predictive analytical tool designed to navigate the randomness of Deriv's synthetic indices. By studying the mathematical relationships between digits — Gatekeepers, Exceedance, Seesaw effects, and real-time entropy — it identifies short statistical windows to help you make informed trading decisions.

## Features
- 🔒 Password-protected lock screen
- 📊 Real-time frequency distribution charts
- 🌀 10×10 digit relationship matrix
- 🔮 Trading Oracle with Matches/Differs signals
- ⚖️ Over/Under trading terminal
- 🚀 Live signal generation with animations

## Markets Supported
- Volatility: R_10, R_25, R_50, R_75, R_100
- Volatility (1s): 1HZ10V, 1HZ25V, 1HZ50V, 1HZ75V, 1HZ100V
- Jump Indices: JD10, JD25, JD50, JD75, JD100
- Daily Random: RDBULL, RDBEAR

## How to Use
1. Open in any modern browser.
2. Enter password
3. Tap "SCAN ALL MARKETS" to analyze all 17 markets.
4. View signal on the Oracle or Over/Under screens.
5. Tap MATCHES/ DIFFERS/ OVER/ UNDER to generate signal.
